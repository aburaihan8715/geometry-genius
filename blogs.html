What is DOM and What is purpose of DOM?

DOM stand for document object model. DOM is not part of javascript.It is part of Browser API. There are lot of Browser API, DOM mont important one of them.DOM is used for manipulating html file.We can easily create, delete, update and styling html element using DOM.

How will you select html elements using DOM?

    We can select DOM by using DOM methods. like...........

    document.getElementById();

    document.getElementsByTagName();

    document.getElementsByClassName();

    document.querySelector();

    document.querySelectorAll();

What is event bubble?

Any action created by user or Browser on the web that is called event, when any event happens it goes to the top or to root element means bubbling and come back to the event creation place means capturing. So we can add handler bubbling time or capturing time.

What is callback function and why will we use it?

Call back means call latter when need. Javascript is synchronous single threaded language means javascript code run line by line and do one task at a time.If we want to take asynchronous action on Javascript callback function come to the table.
